# 🎯 Interview Readiness Analyzer

## 📌 Problem Statement

Design and develop a solution that evaluates different aspects of interview preparation like:

- Technical Skills
- Resume
- Communication Skills
- Portfolio

The system provides:
- Interview Readiness Score
- Personalized Improvement Suggestions

---

## 🚀 Features

✅ Technical Skill Evaluation  
✅ Resume Quality Analysis  
✅ Communication Assessment  
✅ Portfolio Evaluation  
✅ Personalized Suggestions  
✅ Readiness Score Generation

## 📊 Scoring Formula

- Technical Skills → 35%
- Resume → 25%
- Communication → 20%
- Portfolio → 20%
